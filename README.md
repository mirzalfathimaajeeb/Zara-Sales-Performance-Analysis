# 📊 Zara Sales Performance Analysis

An Exploratory Data Analysis (EDA) dashboard built using Power BI as part of my internship at ipsr solutions limited.

## 📝 Project Overview
Analyzed Zara's sales dataset to uncover pricing strategies, promotional impact, and revenue patterns across product segments.

## 🔍 Key Insights
- The £21–£40 price range is Zara's dominant and most profitable segment
- Promoted products drive significantly higher sales volumes
- Seasonal products respond more strongly to promotions
